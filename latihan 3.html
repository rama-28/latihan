Rekayasa Perangkat Lunak

Apa itu Jurusan Rekayasa Perangkat Lunak?

Menurut Muhammad Haziq, Rekayasa Perangkat Lunak adalah sebuah jurusan yang bergerak di bidang perangkat lunak, bisa web, software dan lain lain.
Jurusan RPL digadang gadang menjadi jurusan masa depan karena seiring meningkatnya Dunia digital.
rpl berfokus pada cara-cara pengembangan perangkat lunak (software) termasuk pembuatan, pengembanganan perangkat lunak (software) dan manajemen kualitas

Apa saja yang dipelajari di jurusan RPL?

  1. Dasar Dasar Pengembangan Perangkat Lunak dan Gim
  2. Pemrograman Berorientasi Objek
  3. Pemrograman Web dan Perangkat Bergerak
  4. Pemodelan Perangkat Lunak
  5. Basis Data

Sebetulnya jurusan RPL itu gampang! tidak jauh dari 3 hal ini

  • Coding
  • Desain
  • Agoritma

Apa itu bahasa Pemrograman?

Menurut Wikipedia.org, pemrograman, atau sering diistilahkan juga dengan bahasa komputer atau bahasa pemrograman komputer adalah instruksi standar untuk memerintah komputer.
Bahasa pemrograman ini merupakan suatu himpunan dari aturan sintaks dan semantik yang dipakai untuk mendefinisikan program komputer.
Bahasa ini memungkinkan seorang Programmer dapat menentukan secara persis data mana yang akan diolah oleh komputer, bagaimana data ini akan disimpan/diteruskan, dan jenis langkah apa yang akan diambil dalam berbagai situasi secara persis.

Apa itu coding?

coding adalah aktivitas yang diakukan programmer untuk bisa melakukan komunikasi dengan komputer.

Apa saja bahasa Pemrograman?

Ada banyak bahasa pemrograman,semuanya memiliki fungsi yang berbeda dan syntax yang berbeda pula.
Dibawah ini merupakan contoh bahasa pemrograman:

  • PHP
  • JAVA
  • Python
  • Javascript
  • Kotlin
  • Dan masih banyak lagi
